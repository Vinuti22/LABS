# Мини-каталог — Полноценный сайт (с авторизацией)

**Что добавлено / изменено**

- Простая авторизация (регистрация / вход / выход) — `js/auth.js`.
- Полноценный трёхколоночный интерфейс: панель авторизации, каталог, панель деталей/формы.
- Доработан дизайн в `style.css` (адаптивный, современный вид).
- Разделение на модули: `js/auth.js`, `js/products.js`, `js/storage.js`, `js/ui.js`, `js/main.js`.
- Правила доступа: только авторизованные пользователи могут добавлять/редактировать/удалять товары (на уровне фронтенда).
- Данные хранятся в `localStorage`. Это демо — для реальной системы нужна серверная авторизация и безопасное хранение паролей.

**Примечания по безопасности**

- В демонстрационной реализации пароли хранятся в localStorage с простым хешем — это **не безопасно** для продакшена.
- Для реальной системы используйте HTTPS, серверную аутентификацию, проверку сессий/токенов и защищённое хеширование паролей.
