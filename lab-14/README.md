## Реализованные функции (`utils.js`)

| Функция               | Описание                                                                 | Статус     |
|-----------------------|--------------------------------------------------------------------------|------------|
| `sumOfSquares(arr)`   | Возвращает сумму квадратов всех чисел в массиве<br>Пример: `[1,2,3]` → `1+4+9` = `14` | Работает   |
| `reverseString(str)`  | Переворачивает переданную строку<br>Пример: `"hello"` → `"olleh"`       | Работает   |
| `findMinBroken(arr)`  | Намеренно «сломанная» функция поиска минимального элемента — используется для демонстрации отладки | Содержит ошибки |

### Ошибки в `findMinBroken` (для отладки в DevTools):
1. Инициализация `min = 0` (неправильно — теряем отрицательные числа)
2. Условие сравнения `arr[i] < min` записано с неверным знаком (должно быть `>` при поиске минимума с такой инициализацией, но лучше вообще инициализировать первым элементом)

## Тесты на Jest (`utils.test.js`)

```js
import { sumOfSquares, reverseString, findMinBroken } from './utils.js';

test('сумма квадратов элементов массива', () => {
  expect(sumOfSquares([1, 2, 3])).toBe(14);
  expect(sumOfSquares([-2, 3])).toBe(4 + 9); // 13
  expect(sumOfSquares([])).toBe(0);
});

test('переворот строки', () => {
  expect(reverseString("ABC")).toBe("CBA");
  expect(reverseString("Привет")).toBe("тевирП");
  expect(reverseString("")).toBe("");
});

test('поиск минимума — сломанный вариант (для отладки)', () => {
  expect(findMinBroken([5, 1, -3, 10])).toBe(-3);
  expect(findMinBroken([7, -5, 0])).toBe(-5);
});
